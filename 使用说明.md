# 设计需求平台 — 使用说明

> **文档版本：V1**（与当前线上功能一致；之后功能更新会同步修订文档，版本号顺延 V2、V3……）

## 访问地址

https://isarazyy.github.io/design-platform/

---

## 一、注册与登录

### 注册

1. 打开平台，点击「注册」标签
2. 填写公司邮箱（必须是 @kanyun.com）、密码（至少6位）、姓名
3. 部门：打开企业微信 → 点击头像 → 复制「部门」字段，粘贴到输入框
4. 点击「注册」
5. 去公司邮箱查收验证邮件（注意检查垃圾邮件），点击邮件中的确认链接
6. 回到平台，用邮箱和密码登录

### 登录

1. 输入公司邮箱和密码
2. 点击「登录」

### 忘记密码

1. 在登录页点击「忘记密码？」
2. 输入注册邮箱，点击「发送重置邮件」
3. 去邮箱点击重置链接，设置新密码

---

## 二、首页

登录后进入首页，页面上方有统计卡片：全部需求 / 待制作 / 制作中 / 待审核 / 已交付。

**点击统计卡片可以直接筛选**：比如点「制作中」，下方列表就只显示制作中的需求；再点一次取消筛选。选中的卡片会有高亮边框。

### 筛选需求

| 操作 | 说明 |
|------|------|
| 点击统计卡片 | 快速筛选对应状态，再点一次取消 |
| 搜索框 | 输入关键词，可搜标题、提需人、部门 |
| 需求类型下拉 | 筛选视觉设计/美术/动画 |
| 全部 / 我创建的 / 我参与的 | 切换查看范围 |

### 删除需求

- 在需求卡片右侧点击垃圾桶图标
- 只有需求的创建者和管理员能看到删除按钮

---

## 三、创建需求

点击首页右侧的「创建需求」按钮，按以下步骤填写：

### 第一步：基本信息（必填）
- **需求标题**：简要描述这个需求是做什么的
- **提需人**：会自动填入你的姓名
- **部门**：会自动填入你注册时填的部门，也可以重新从企业微信复制
- **需求类型**：选择视觉设计 / 美术 / 动画

### 第二步：期望交付日期 & 优先级（选填）
- 选择开始日期和截止日期（截止日期不能早于开始日期）
- 选择优先级：紧急 / 高 / 中 / 低
- **版本提交节点**：点「添加版本」后出现 **V1、V2…**（自动编号），为每个版本填写提交日期（须在开始～截止日期之间）和备注；删除某行后序号会顺延重排

### 第三步：需求背景 & 目标（选填）
- 说明为什么要做这个需求
- 希望达成什么效果

### 第四步：需求物料 & 文案
点击「添加物料」，每个物料是一个独立的卡片：

| 字段 | 说明 |
|------|------|
| 物料类型 | 下拉选择：长图、H5、海报、Banner、插画、icon、视频、动画、包装、礼品、绘本、其他 |
| 尺寸/规格 | 下拉选择常见尺寸，或直接输入自定义尺寸 |
| 数量 | 这种物料要做几张/几个 |
| 文案内容 | 选择「模版」（主标题/副标题/正文）或「自由文本」 |

每个物料独立填写文案，适合不同物料需要不同文案的情况（比如两张海报文案不同）。

### 第五步：设计要求 & 参考素材（选填）
- 写设计要求说明（配色、字体、风格等）
- 上传参考图片（拖拽或点击上传，支持多张）
- 添加参考链接

### 第六步：补充说明（选填）
其他需要补充的内容

### 提交
- 点击右下角「提交需求」
- 也可以点击「保存草稿」，下次从草稿箱继续编辑
- 点击「清空」会清除所有已填内容

> 平台每5秒自动保存一次草稿，即使浏览器意外关闭也不会丢失。

---

## 四、草稿箱

点击侧边栏「草稿箱」进入：

- 查看所有保存的草稿
- 点击草稿卡片继续编辑
- 点击删除按钮清除不需要的草稿
- 提交需求后草稿自动清除

支持同时保存多个草稿，方便在多个需求之间切换。

---

## 五、需求详情

点击任意需求进入详情页，可以做以下操作：

### 查看内容
- 需求的所有信息都在详情页展示
- 物料和文案合并展示，每个物料下方直接显示对应文案
- 参考图片可以点击放大查看

### 编辑需求
- 点击右上角「编辑」按钮（需要有编辑权限）
- 创建者、协作人、管理员可以编辑

### 变更状态
状态流程：

```
待制作 → 制作中 → 待审核 → 已交付
```

- 每个状态下方有对应的操作按钮（如「开始制作」「提交审核」「审核通过，交付」）
- 创建者、被指派的负责人、管理员可以变更状态

### 对接方（负责人与协作人）
- 在「对接方」区域操作
- **负责人**：必须从已注册用户中选择；在框内**输入姓名或邮箱**可快速筛选列表
- **协作人**：可以一起编辑这个需求的同事，可从列表选或直接输入姓名
- 创建者和管理员可以操作

### 分享
- 点击「复制链接」把需求地址发给同事
- 点击「打印」可以打印需求单

### 修改记录
- 页面底部显示所有操作记录（谁在什么时候做了什么）

---

## 六、管理后台（仅管理员）

管理员在侧边栏点击「管理」进入。

功能：
- 查看所有注册用户
- 修改用户角色（普通用户 / 管理员）
- 不能修改自己的角色

| 删除自己创建的需求 | ✓ | ✓ |
