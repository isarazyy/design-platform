# 设计需求平台 — 使用说明

## 访问地址

https://isarazyy.github.io/design-platform/

---

## 一、注册与登录

### 注册

1. 打开平台，点击「注册」标签
2. 填写公司邮箱（必须是 @kanyun.com）、密码（至少6位）、姓名、部门
3. 点击「注册」
4. 注册成功后切换到「登录」标签，用刚才的邮箱密码登录

### 登录

1. 输入公司邮箱和密码
2. 点击「登录」

---

## 二、首页

登录后进入首页，页面上方有四个统计卡片：全部需求 / 待审核 / 设计中 / 已交付。

**点击统计卡片可以直接筛选**：比如点「设计中」，下方列表就只显示设计中的需求；再点一次取消筛选。选中的卡片会有高亮边框。

### 筛选需求

| 操作 | 说明 |
|------|------|
| 点击统计卡片 | 快速筛选对应状态，再点一次取消 |
| 搜索框 | 输入关键词，可搜标题、提需人、部门 |
| 需求类型下拉 | 筛选视觉设计/美术/动画 |
| 状态下拉 | 筛选待审核/设计中/已交付/已关闭 |
| 全部 / 我创建的 / 我参与的 | 切换查看范围 |

### 删除需求

- 在需求卡片右侧点击垃圾桶图标
- 只有需求的创建者和管理员能看到删除按钮

---

## 三、创建需求

点击首页右侧的「创建需求」按钮，按以下步骤填写：

### 第一步：基本信息（必填）
- **需求标题**：简要描述这个需求是做什么的
- **提需人**：会自动填入你的姓名
- **部门**：会自动填入你注册时填的部门
- **需求类型**：选择视觉设计 / 美术 / 动画

### 第二步：需求背景 & 目标（选填）
- 说明为什么要做这个需求
- 希望达成什么效果

### 第三步：日期与优先级（选填）
- 选择开始日期和截止日期
- 选择优先级：紧急 / 高 / 中 / 低

### 第四步：需求物料
点击「添加物料」，每一行代表一种物料：

| 字段 | 说明 |
|------|------|
| 物料类型 | 下拉选择：长图、H5、海报、Banner、插画、icon、视频、动画、包装、礼品、绘本、其他 |
| 尺寸/规格 | 下拉选择常见尺寸，或手动输入自定义尺寸 |
| 数量 | 这种物料要做几张/几个 |

可以添加多行，每行一种。

### 第五步：需求文案（选填）
两种模式：
- **使用模版**：分别填主标题、副标题、正文
- **自由文本**：一个文本框自由写

### 第六步：设计要求 & 参考素材（选填）
- 写设计要求说明（配色、字体、风格等）
- 上传参考图片（拖拽或点击上传，支持多张）
- 添加参考链接

### 第七步：补充说明（选填）
其他需要补充的内容

### 提交
- 点击右下角「提交需求」
- 也可以点击「保存草稿」，下次打开时自动恢复
- 点击「清空」会清除所有已填内容

> 平台每5秒自动保存一次草稿，即使浏览器意外关闭也不会丢失。

---

## 四、需求详情

点击任意需求进入详情页，可以做以下操作：

### 查看内容
- 需求的所有信息都在详情页展示
- 参考图片可以点击放大查看

### 编辑需求
- 点击右上角「编辑」按钮（需要有编辑权限）
- 创建者、协作人、管理员可以编辑

### 变更状态
- 右上角有状态下拉框
- 状态只能按流程走：

```
待审核 → 设计中 → 已交付 → 已关闭
         ↘ 已关闭  ↘ 已关闭
```

- 不能跳步，比如不能从「待审核」直接跳到「已交付」
- 「已关闭」后不能再变更
- 创建者、设计师、管理员可以变更状态

### 指派制作人与协作人
- 在「制作人与协作」区域操作
- **制作人**：负责制作的设计师，从下拉框选择
- **协作人**：可以一起编辑这个需求的同事，可选多人
- 创建者和管理员可以操作

### 分享
- 点击「复制链接」把需求地址发给同事
- 点击「打印」可以打印需求单

### 修改记录
- 页面底部显示所有操作记录（谁在什么时候做了什么）

---

## 五、管理后台（仅管理员）

管理员在页面右上角头像菜单中点击「管理」进入。

功能：
- 查看所有注册用户
- 修改用户角色（普通用户 / 设计师 / 管理员）
- 不能修改自己的角色
