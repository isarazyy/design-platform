设计需求平台 — 产品介绍

文档版本：V1（当前对外说明与使用手册以此版为准；后续产品迭代将依次更新为 V2、V3……）

一句话说明

设计需求平台是看云内部使用的设计需求提交与管理工具，让需求方和制作人在一个地方完成从「提需求」到「交付」的全流程协作。

解决什么问题

以前 现在
需求散落在微信、邮件、口头沟通中 统一在平台上提交，所有人看到的信息一致
物料要什么尺寸、做几张说不清楚 每条物料单独填写类型、尺寸、数量，并附带对应文案
不知道需求进展到哪一步了 状态实时更新：待制作 → 制作中 → 待审核 → 已交付
谁改了什么、什么时候改的没记录 每次操作自动记录修改日志
想找之前的需求翻聊天记录 搜索、筛选、分页，历史需求随时查
同时写多个需求来回切换很麻烦 草稿箱支持保存多个草稿，随时继续编辑

核心功能

1. 账号与安全

  • 公司邮箱（@kanyun.com）注册登录
  • 邮箱验证：注册后需点击验证邮件才能登录，防止伪造账号
  • 忘记密码：通过邮箱重置密码
  • 两种角色：普通用户 / 管理员
  • 管理员可在后台修改用户角色

2. 创建需求

  • 基本信息：标题、提需人、部门（从企业微信复制）、需求类型（视觉设计/美术/动画）
  • 期望交付日期与优先级（紧急/高/中/低），截止日期不能早于开始日期；版本提交节点（V1/V2…各对应提交日期与备注，自动编号）
  • 需求背景与目标
  • 需求物料 & 文案：每个物料独立填写类型、尺寸（可选或自由输入）、数量，以及对应的文案内容（模版或自由文本）
  • 设计要求说明、参考图片上传、参考链接
  • 补充说明
  • 自动保存草稿，断网不丢

3. 草稿箱

  • 支持同时保存多个草稿
  • 侧边栏「草稿箱」入口，查看所有草稿
  • 点击草稿继续编辑，提交后自动清除

4. 需求管理

  • 首页统计面板：全部需求 / 待制作 / 制作中 / 待审核 / 已交付，点击卡片直接筛选
  • 快速筛选：按类型、关键词搜索
  • 我的需求：「全部」「我创建的」「我参与的」三个视角切换
  • 逾期标红提醒
  • 分页浏览，每页20条

5. 需求详情与协作

  • 查看完整需求内容（物料与文案合并展示）
  • 指派负责人（从注册用户中搜索选择）
  • 添加协作人（协作人可编辑需求，支持自由输入姓名）
  • 状态流转进度条 + 操作按钮：待制作 → 制作中 → 待审核 → 已交付
  • 复制链接分享给同事
  • 打印需求单
  • 修改记录时间线

6. 权限控制

  • 创建者和管理员可以删除需求
  • 创建者、协作人、管理员可以编辑需求
  • 创建者、指派的制作人、管理员可以变更状态
  • 创建者和管理员可以指派制作人和协作人

设计需求平台 — 使用说明

文档版本：V1（与当前线上功能一致；之后功能更新会同步修订文档，版本号顺延 V2、V3……）

访问地址

https://isarazyy.github.io/design-platform/

一、注册与登录

注册

  1. 打开平台，点击「注册」标签
  2. 填写公司邮箱（必须是 @kanyun.com）、密码（至少6位）、姓名
  3. 部门：打开企业微信 → 点击头像 → 复制「部门」字段，粘贴到输入框
  4. 点击「注册」
  5. 去公司邮箱查收验证邮件（注意检查垃圾邮件），点击邮件中的确认链接
  6. 回到平台，用邮箱和密码登录

登录

  1. 输入公司邮箱和密码
  2. 点击「登录」

忘记密码

  1. 在登录页点击「忘记密码？」
  2. 输入注册邮箱，点击「发送重置邮件」
  3. 去邮箱点击重置链接，设置新密码

二、首页

登录后进入首页，页面上方有统计卡片：全部需求 / 待制作 / 制作中 / 待审核 / 已交付。

点击统计卡片可以直接筛选：比如点「制作中」，下方列表就只显示制作中的需求；再点一次取消筛选。选中的卡片会有高亮边框。

筛选需求

操作 说明
点击统计卡片 快速筛选对应状态，再点一次取消
搜索框 输入关键词，可搜标题、提需人、部门
需求类型下拉 筛选视觉设计/美术/动画
全部 / 我创建的 / 我参与的 切换查看范围

删除需求

  • 在需求卡片右侧点击垃圾桶图标
  • 只有需求的创建者和管理员能看到删除按钮

三、创建需求

点击首页右侧的「创建需求」按钮，按以下步骤填写：

第一步：基本信息（必填）

  • 需求标题：简要描述这个需求是做什么的
  • 提需人：会自动填入你的姓名
  • 部门：会自动填入你注册时填的部门，也可以重新从企业微信复制
  • 需求类型：选择视觉设计 / 美术 / 动画

第二步：期望交付日期 & 优先级（选填）

  • 选择开始日期和截止日期（截止日期不能早于开始日期）
  • 选择优先级：紧急 / 高 / 中 / 低
  • 版本提交节点：点「添加版本」后出现 V1、V2…（自动编号），为每个版本填写提交日期（须在开始～截止日期之间）和备注；删除某行后序号会顺延重排

第三步：需求背景 & 目标（选填）

  • 说明为什么要做这个需求
  • 希望达成什么效果

第四步：需求物料 & 文案

点击「添加物料」，每个物料是一个独立的卡片：

字段 说明
物料类型 下拉选择：长图、H5、海报、Banner、插画、icon、视频、动画、包装、礼品、绘本、其他
尺寸/规格 下拉选择常见尺寸，或直接输入自定义尺寸
数量 这种物料要做几张/几个
文案内容 选择「模版」（主标题/副标题/正文）或「自由文本」

每个物料独立填写文案，适合不同物料需要不同文案的情况（比如两张海报文案不同）。

第五步：设计要求 & 参考素材（选填）

  • 写设计要求说明（配色、字体、风格等）
  • 上传参考图片（拖拽或点击上传，支持多张）
  • 添加参考链接

第六步：补充说明（选填）

其他需要补充的内容

提交

  • 点击右下角「提交需求」
  • 也可以点击「保存草稿」，下次从草稿箱继续编辑
  • 点击「清空」会清除所有已填内容

平台每5秒自动保存一次草稿，即使浏览器意外关闭也不会丢失。

四、草稿箱

点击侧边栏「草稿箱」进入：

  • 查看所有保存的草稿
  • 点击草稿卡片继续编辑
  • 点击删除按钮清除不需要的草稿
  • 提交需求后草稿自动清除

支持同时保存多个草稿，方便在多个需求之间切换。

五、需求详情

点击任意需求进入详情页，可以做以下操作：

查看内容

  • 需求的所有信息都在详情页展示
  • 物料和文案合并展示，每个物料下方直接显示对应文案
  • 参考图片可以点击放大查看

编辑需求

  • 点击右上角「编辑」按钮（需要有编辑权限）
  • 创建者、协作人、管理员可以编辑

变更状态

状态流程：

待制作 → 制作中 → 待审核 → 已交付
  • 每个状态下方有对应的操作按钮（如「开始制作」「提交审核」「审核通过，交付」）
  • 创建者、被指派的负责人、管理员可以变更状态

对接方（负责人与协作人）

  • 在「对接方」区域操作
  • 负责人：必须从已注册用户中选择；在框内输入姓名或邮箱可快速筛选列表
  • 协作人：可以一起编辑这个需求的同事，可从列表选或直接输入姓名
  • 创建者和管理员可以操作

分享

  • 点击「复制链接」把需求地址发给同事
  • 点击「打印」可以打印需求单

修改记录

  • 页面底部显示所有操作记录（谁在什么时候做了什么）

六、管理后台（仅管理员）

管理员在侧边栏点击「管理」进入。

功能： - 查看所有注册用户 - 修改用户角色（普通用户 / 管理员） - 不能修改自己的角色
